    **Step 4: Create Makefile in project root.**
    Targets:
    - `install`: runs `uv sync` then `cd web && pnpm install`
    - `dev-api`: runs `uvicorn src.api.app:app --reload --port 8000`
    - `dev-web`: runs `cd web && pnpm dev`
    - `dev`: runs both via `npx concurrently --names "api,web" --prefix-colors "blue,green" "uvicorn src.api.app:app --reload --port 8000" "cd web && pnpm dev"`
    - `test`: runs `python -m pytest tests/ -x -q`
    - `test-api`: runs `python -m pytest tests/test_api_summaries.py -x -q`
    All targets should be .PHONY.

    **Step 5: Add concurrently to web/package.json devDependencies.**
    Run `cd web && pnpm add -D concurrently` so npx concurrently works from the Makefile dev target.
    Update the Makefile dev target to use `cd web && npx concurrently ...` instead of bare `npx concurrently` so it finds the local install.

    **Step 6: Ensure next.config.ts does not conflict.**
    Check the generated next.config.ts. It should be minimal. No rewrites or proxies needed -- the page fetches directly from localhost:8000 with CORS.
