**providers.tsx**: Import `ThemeProvider` from `next-themes`. Wrap the entire content (including QueryClientProvider) with ThemeProvider:
```
<ThemeProvider attribute="class" defaultTheme="system" enableSystem disableTransitionOnChange storageKey="theme">
  <QueryClientProvider ...>
    {children}
    ...
  </QueryClientProvider>
</ThemeProvider>
```
The `attribute="class"` makes next-themes add `.dark` to the `<html>` element, which the existing `@custom-variant dark` directive in globals.css already targets. `disableTransitionOnChange` prevents flash during theme switch.

**layout.tsx**: Add `suppressHydrationWarning` to the `<html>` element to prevent React hydration mismatch errors (next-themes injects an inline script that sets the class before hydration):
```
<html lang="en" suppressHydrationWarning className={...}>
```
No other changes to layout.tsx.
