<task type="auto">
  <name>Task 1: Markdown typography overhaul</name>
  <files>web/src/components/summary/markdown-renderer.tsx, web/src/app/globals.css</files>
  <action>
Rewrite the markdown component overrides in markdown-renderer.tsx for a premium reading experience. The user reads summaries daily -- this is the highest-impact area.

**Heading hierarchy:**
- `h1`: `text-xl font-semibold tracking-tight mt-8 mb-4` (rarely used but should be distinct)
- `h2`: Keep sticky behavior. Increase to `text-lg font-semibold tracking-tight`. Add a subtle bottom border (`border-b border-border`). Add `mt-8 mb-3 pb-2` for breathing room between sections. The sticky background should match the page background precisely.
- `h3`: `text-base font-medium mt-6 mb-2` -- clear step down from h2 but still distinct from body text
- `h4`: `text-sm font-medium mt-4 mb-1.5` -- rare but should be styled

**Body text:**
- `p`: `text-sm leading-relaxed mb-4` -- increase bottom margin from mb-3 for more rhythm between paragraphs
- `strong`: ensure it renders with `font-semibold` (may need explicit override)

**Lists:**
- `ul`: `mb-4 space-y-1.5 pl-5 text-sm` -- add `space-y-1.5` for breathing room between items
- `ol`: same spacing as ul
- `li`: `text-sm leading-relaxed` -- keep, but add `pl-1` for slight indent from bullet. Nested lists should have `mt-1.5` gap.

**Code:**
- Inline `code`: `rounded-md bg-muted px-1.5 py-0.5 text-[13px] font-mono` -- slightly larger than xs, use the mono font variable
- Block `pre`: `mb-4 rounded-lg border border-border bg-muted/50 p-4 overflow-x-auto` -- add border for definition, rounded-lg for modern feel
- Block `code` (inside pre): `text-[13px] font-mono leading-relaxed` -- no background (pre handles it)

**Blockquote:**
- `border-l-2 border-primary/40 bg-primary/5 py-2 pl-4 mb-4 text-sm italic text-muted-foreground` -- use brand green tint instead of generic muted

**Tables:**
- Table wrapper: `mb-4 overflow-x-auto rounded-lg border border-border`
- `th`: `bg-muted/50 px-4 py-2.5 text-left text-xs font-medium uppercase tracking-wider` -- uppercase headers, more padding
- `td`: `px-4 py-2.5 text-sm border-t border-border` -- use border-t instead of full borders for cleaner look

**Links:**
- `text-primary underline decoration-primary/30 hover:decoration-primary transition-colors` -- subtle underline that strengthens on hover

**Horizontal rule (hr):**
- `my-8 border-t border-border` -- generous spacing

**globals.css**: Add a `.markdown-content` scoped style block for any CSS that is difficult to express as Tailwind classes, e.g.:
```css
.markdown-content > *:first-child { margin-top: 0; }
.markdown-content > *:last-child { margin-bottom: 0; }
```
Also ensure dark mode scrollbar styling works (`.dark .scrollbar-thin` rules).
