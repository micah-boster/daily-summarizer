  1. Create `src/sidecar.py` with Pydantic models:

     ```python
     class SidecarTask(BaseModel):
         description: str
         owner: str | None = None
         source_meeting: str
         date_captured: str  # ISO date
         due_date: str | None = None
         status: str = "new"  # new, in-progress, completed

     class SidecarDecision(BaseModel):
         description: str
         decision_makers: list[str] = Field(default_factory=list)
         rationale: str | None = None
         source_meeting: str

     class SidecarMeeting(BaseModel):
         title: str
         time: str  # ISO datetime
         participants: list[str] = Field(default_factory=list)
         has_transcript: bool = False

     class DailySidecar(BaseModel):
         date: str  # ISO date
         generated_at: str  # ISO datetime
         meeting_count: int
         transcript_count: int
         tasks: list[SidecarTask] = Field(default_factory=list)
         decisions: list[SidecarDecision] = Field(default_factory=list)
         source_meetings: list[SidecarMeeting] = Field(default_factory=list)
     ```

  2. Add `build_daily_sidecar()` function:
     ```python
     def build_daily_sidecar(
         synthesis: DailySynthesis,
         extractions: list[MeetingExtraction],
     ) -> DailySidecar:
     ```
     Implementation:
     - `date` = synthesis.date.isoformat()
     - `generated_at` = synthesis.generated_at.isoformat()
     - `meeting_count` = synthesis.meeting_count
     - `transcript_count` = synthesis.transcript_count
     - For each non-low-signal extraction:
       - Map `commitments` to `SidecarTask`:
         - `description` = item.content
         - `owner` = first participant if available, else None
         - `source_meeting` = extraction.meeting_title
         - `date_captured` = synthesis.date.isoformat()
         - `due_date` = None (not reliably parseable from extraction text)
         - `status` = "new"
       - Map `decisions` to `SidecarDecision`:
         - `description` = item.content
         - `decision_makers` = item.participants
         - `rationale` = item.rationale
         - `source_meeting` = extraction.meeting_title
       - Add meeting to `source_meetings`:
         - `title` = extraction.meeting_title
         - `time` = extraction.meeting_time
         - `participants` = extraction.meeting_participants
         - `has_transcript` = True (it has an extraction)
     - Also add meetings WITHOUT transcripts to source_meetings (from synthesis.meetings_without_transcripts):
       - `has_transcript` = False
       - `title` = event.title, `time` = event.start_time.isoformat() if set
       - `participants` = [a.name or a.email for a in event.attendees if not a.is_self]
