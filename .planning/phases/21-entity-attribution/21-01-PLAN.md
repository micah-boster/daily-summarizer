**GREEN phase -- implement src/entity/attributor.py:**

Create a module with these functions (NO class -- matches project style of discovery.py, normalizer.py):

- `content_hash(text: str) -> str` -- `hashlib.sha256(text.encode("utf-8")).hexdigest()[:16]`

- `commitment_content_hash(who: str, what: str, by_when: str) -> str` -- `content_hash(f"{who}|{what}|{by_when}")`

- `match_name_to_entity(raw_name: str, repo: EntityRepository) -> tuple[Entity, float] | None` -- Two-step matching per user decision:
  1. Try `repo.get_by_name(normalize_for_matching(raw_name))` then `repo.get_by_name(raw_name)` -- confidence 1.0
  2. Fall back to direct alias SQL query: `SELECT a.entity_id FROM aliases a JOIN entities e ON a.entity_id = e.id WHERE LOWER(a.alias) = LOWER(?) AND e.deleted_at IS NULL` -- confidence 0.7
  3. Follow merge_target_id if present via `repo.get_by_id(entity.merge_target_id)`
  4. Return None if nothing matches

- `attribute_synthesis_items(synthesis_output: DailySynthesisOutput, repo: EntityRepository, target_date: str) -> AttributionResult` -- Iterate over substance, decisions, commitments. For each item, build candidate name set from `entity_names`. For CommitmentRow, also add `who` (excluding empty/"TBD"/"tbd"). Match each candidate via `match_name_to_entity`. Build per-item EntityReference list and EntityMention list. Return an `AttributionResult` dataclass with: `substance_refs: list[list[EntityReference]]`, `decision_refs: list[list[EntityReference]]`, `commitment_refs: list[list[EntityReference]]`, `mentions: list[EntityMention]`, `entity_summary: list[EntitySummaryEntry]`.

- `persist_mentions(conn: sqlite3.Connection, mentions: list[EntityMention], target_date: str) -> int` -- DELETE FROM entity_mentions WHERE source_date = target_date, then INSERT each mention. Commit. Return count.

Define Pydantic models in attributor.py:
- `EntityReference(BaseModel)`: entity_id (str), name (str), confidence (float)
- `EntitySummaryEntry(BaseModel)`: entity_id (str), name (str), entity_type (str), mention_count (int)
- `AttributionResult(BaseModel)`: substance_refs (list[list[EntityReference]]), decision_refs (list[list[EntityReference]]), commitment_refs (list[list[EntityReference]]), mentions (list[EntityMention]), entity_summary (list[EntitySummaryEntry])
