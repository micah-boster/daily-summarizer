   - commandPaletteOpen: boolean (default false)
   - toggleCommandPalette: () => void
   - recentEntities: string[] (entity IDs, max 5, persisted)
   - recentDates: string[] (date strings, max 5, persisted)
   - addRecentEntity: (entityId: string) => void
   - addRecentDate: (date: string) => void
   - Update selectEntity to also call addRecentEntity

2. Create command-palette.tsx using shadcn Command (wraps cmdk):
   - Use CommandDialog (Command inside a Dialog) for the overlay
   - Global keyboard listener: Cmd+K (metaKey) or Ctrl+K (ctrlKey) toggles open state. Prevent default browser behavior. Do NOT bind when a textarea or contenteditable is focused.

   **Before typing (empty input) -- show recents:**
   - CommandGroup "Recent Entities": list recentEntities from store, show entity name + type badge
   - CommandGroup "Recent Dates": list recentDates from store, formatted nicely
   - If no recents: show a subtle hint "Start typing to search entities, dates, or actions..."

   **While typing -- filtered results in three groups:**

   CommandGroup "Entities" (icon: user/building):
   - Fetch entities from GET /entities (can use cached query data from TanStack Query)
   - Filter by fuzzy match against entity name and aliases (cmdk handles fuzzy matching)
   - Show entity name + type badge + mention count
   - On select: close palette, set activeTab to "entities", selectEntity(id), navigate to entity view

   CommandGroup "Dates" (icon: calendar):
   - Parse common date formats from input: "2026-04-08", "April 8", "yesterday", "last Monday"
   - For recognized dates: show formatted date with available summary indicator
   - On select: close palette, set activeTab to "summaries", navigate to that date's summary

   CommandGroup "Actions" (icon: zap/bolt):
   - Static list of actions:
     - "Create Entity" -- opens form panel in create mode
     - "Review Merge Proposals" -- navigates to merge review
   - On select: close palette, execute the action

   **Prefix filters (power user shortcuts):**
   - Input starts with "/": show only Actions group
   - Input starts with "@": show only Entities group (strip @ for matching)
   - Input starts with "#": show only Dates group (strip # for matching)

   **Recency boost within groups:**
   - Entities that appear in recentEntities should sort to top of Entities group
   - Dates that appear in recentDates should sort to top of Dates group

3. Styling:
   - Centered overlay (Dialog), similar width to Linear's Cmd+K (~640px)
   - Clean, minimal design with group headers
   - Keyboard hint in footer: "↑↓ Navigate  ↵ Select  esc Close"
   - Input placeholder: "Search entities, dates, or actions..."
