**Add to `src/entity/views.py`:**

```python
def generate_entity_report(
    repo: EntityRepository,
    entity_name: str,
    from_date: date | None = None,
    to_date: date | None = None,
    output_dir: str = "output/entities",
    template_dir: str = "templates",
) -> Path:
```

Implementation:
1. Call `get_entity_scoped_view(repo, entity_name, from_date, to_date)` to get view data
2. Look up aliases for the entity via `repo.list_aliases(view.entity_id)`
3. Load Jinja2 template: `Environment(loader=FileSystemLoader(template_dir)).get_template("entity_report.md.j2")`
4. Render template with view data + aliases + generated_date (today's date)
5. Create output_dir if not exists (Path(output_dir).mkdir(parents=True, exist_ok=True))
6. Write to `{output_dir}/{entity_name_slug}.md` where slug is entity name lowered with spaces replaced by hyphens
7. Return the Path to the written file

**Add `_cmd_report` handler:**
```python
def _cmd_report(repo, args, config):
    from src.entity.views import generate_entity_report
    output_dir = getattr(args, "output_dir", None) or str(Path(config.pipeline.output_dir) / "entities")
    try:
        path = generate_entity_report(
            repo=repo,
            entity_name=args.name,
            from_date=getattr(args, "from_date", None),
            to_date=getattr(args, "to_date", None),
            output_dir=output_dir,
        )
        print("Report generated: %s" % path)
    except ValueError as e:
        print("Error: %s" % e, file=sys.stderr)
        sys.exit(1)
```
