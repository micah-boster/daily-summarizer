**Create src/ingest/normalizer.py:**

1. `match_transcript_to_event(transcript: dict, events: list[NormalizedEvent], time_window_minutes: int = 30) -> NormalizedEvent | None` — Find the best matching calendar event for a transcript:
   - For each event with a start_time, compute time difference in minutes: `abs((transcript["meeting_time"] - event.start_time).total_seconds() / 60)`
   - Only consider events within `time_window_minutes`
   - Compute title similarity using `difflib.SequenceMatcher(None, transcript["title"].lower(), event.title.lower()).ratio()`
   - Score candidates: sort by title similarity (desc), then time proximity (asc)
   - Return the best match, or None if no candidates
   - IMPORTANT: Normalize both datetimes to the same timezone before comparison. If one is naive and one is aware, make both aware using the configured timezone.

2. `match_transcripts_to_events(transcripts: list[dict], events: list[NormalizedEvent], config: dict) -> tuple[list[NormalizedEvent], list[dict]]` — Batch matching:
   - Read `time_window_minutes` from `config["transcripts"]["matching"]`
   - For each transcript, find best matching event
   - When a match is found, set `event.transcript_text = transcript["transcript_text"]` and `event.transcript_source = transcript["source"]`
   - When both Gemini and Gong match the same event: Gemini wins (per user decision). If event already has a transcript with source="gemini", skip the gong match. If event has source="gong" and new match is source="gemini", overwrite.
   - Collect unmatched transcripts (those with no event match)
   - Return (updated_events, unmatched_transcripts)
   - Log: "Matched {N}/{M} transcripts to calendar events. {K} unmatched."

3. `deduplicate_events(events: list[NormalizedEvent]) -> list[NormalizedEvent]` — Remove duplicate calendar events:
   - Group events by a dedup key: `(title.lower().strip(), start_time)` for timed events, `(title.lower().strip(), date)` for all-day events
   - For duplicates: keep the one with the most data (has transcript > no transcript, has attendees > no attendees)
   - Log: "Deduplicated {N} events down to {M}"
   - Return deduplicated list

4. `build_normalized_output(categorized: dict, transcripts: list[dict], config: dict) -> tuple[dict, list[dict]]` — Top-level normalization:
   - Combine timed_events + all_day_events into single list for matching
   - Call `match_transcripts_to_events`
   - Call `deduplicate_events` on matched events
   - Re-categorize into timed/all_day after dedup
   - Return (updated_categorized_dict, unmatched_transcripts)

All functions: type hints, docstrings, logging.
