  <action>
  1. Create `src/dedup.py`:
     ```python
     """Algorithmic cross-source dedup pre-filter.

     Consolidates near-identical items across sources before LLM synthesis.
     Uses title similarity within the same calendar day as the matching signal.
     Conservative threshold to minimize false positives.
     """
     ```

     Implement `dedup_source_items(items: list[SourceItem], config: PipelineConfig, target_date: date) -> list[SourceItem]`:
     - If `config.dedup.enabled` is False, return items unchanged.
     - If items is empty, return empty list.
     - Group items by calendar day (using `item.timestamp.date()`). Only process items matching `target_date`.
     - Within the target day group, compare all pairs using `difflib.SequenceMatcher` on lowercased, stripped titles.
     - If similarity ratio >= `config.dedup.similarity_threshold`, mark as duplicates.
     - Use union-find / greedy clustering: build clusters of mutually-similar items.
     - For each cluster with >1 item, merge:
       - Keep the item with the longest `content` as the base.
       - Combine `display_context` from all items: e.g., "Slack #general, Notion Q2 Planning".
       - Union all `participants` lists (deduplicated, sorted).
       - Keep the earliest `timestamp`.
       - Append a `context["dedup_sources"]` list with the ids and source_types of all merged items.
     - Log each merge decision to a separate file in `config.dedup.log_dir`:
       - File name: `dedup_{target_date.isoformat()}.log`
       - Format per line: `"Merged '{title_a}' ({source_a}) + '{title_b}' ({source_b}) -- score: {ratio:.2f}"`
       - Create the log_dir if it doesn't exist.
     - Return the deduplicated list (merged items + non-duplicate items).

  2. In `src/pipeline.py`:
     - Add import: `from src.dedup import dedup_source_items`
     - After all ingest calls return (after `notion_items = _ingest_notion(ctx)`) and before synthesis, add:
       ```python
       # Cross-source dedup pre-filter
       all_source_items = (slack_items or []) + (hubspot_items or []) + (docs_items or []) + (notion_items or [])
       try:
           deduped_items = dedup_source_items(all_source_items, ctx.config, ctx.target_date)
           # Redistribute back to per-source lists for synthesis compatibility
           deduped_slack = [i for i in deduped_items if i.source_type in (SourceType.SLACK_MESSAGE, SourceType.SLACK_THREAD)]
           deduped_hubspot = [i for i in deduped_items if i.source_type in (SourceType.HUBSPOT_DEAL, SourceType.HUBSPOT_CONTACT, SourceType.HUBSPOT_TICKET, SourceType.HUBSPOT_ACTIVITY)]
           deduped_docs = [i for i in deduped_items if i.source_type in (SourceType.GOOGLE_DOC_EDIT, SourceType.GOOGLE_DOC_COMMENT)]
           deduped_notion = [i for i in deduped_items if i.source_type in (SourceType.NOTION_PAGE, SourceType.NOTION_DB)]
           items_removed = len(all_source_items) - len(deduped_items)
           if items_removed:
               logger.info("Dedup pre-filter: consolidated %d items", items_removed)
           slack_items = deduped_slack
           hubspot_items = deduped_hubspot
           docs_items = deduped_docs
           notion_items = deduped_notion
       except Exception as e:
           logger.warning("Dedup pre-filter failed: %s. Continuing with unfiltered items.", e)
       ```
     - Add import: `from src.models.sources import SourceType` (if not already imported).
