       - `WeeklyRollupResponse`: `{ week_number: int; year: int; start_date: string; end_date: string; markdown: string; sidecar: object | null }`
       - `MonthlyRollupListItem`: `{ month_label: string; year: int; month: int }`
       - `MonthlyRollupResponse`: `{ month: int; year: int; markdown: string; sidecar: object | null }`

    4. Create `web/src/stores/ui-store.ts` — Zustand store:
       - `leftNavCollapsed: boolean` (default false)
       - `rightSidebarCollapsed: boolean` (default false)
       - `collapsedSections: Record<string, boolean>` (for card collapse state)
       - `expandedNavGroups: Record<string, boolean>` (for Daily/Weekly/Monthly nav groups, default all true)
       - Toggle functions for each
       - Use `persist` middleware with `createJSONStorage(() => sessionStorage)` and store name `"ui-state"`

    5. Create `web/src/hooks/use-summaries.ts` — TanStack Query hooks:
       - `useSummaryList()` — fetches `/summaries`, queryKey: `["summaries"]`
       - `useSummary(date: string | null)` — fetches `/summaries/${date}`, queryKey: `["summary", date]`, enabled only when date is truthy
       - `useWeeklyList()` — fetches `/summaries/weekly`, queryKey: `["weekly-rollups"]`
       - `useWeeklyRollup(year: number, week: number)` — fetches `/summaries/weekly/${year}/${week}`, queryKey: `["weekly-rollup", year, week]`
       - `useMonthlyList()` — fetches `/summaries/monthly`, queryKey: `["monthly-rollups"]`
       - `useMonthlyRollup(year: number, month: number)` — fetches `/summaries/monthly/${year}/${month}`, queryKey: `["monthly-rollup", year, month]`
       - `useStatus()` — fetches `/status`, queryKey: `["status"]`
       - All queries use `staleTime: 5 * 60 * 1000`

    6. Create `web/src/app/providers.tsx` — marked "use client":
