  <action>
1. Create `src/ingest/__init__.py` (empty).

2. Create `src/ingest/calendar.py` with:

   - `def build_calendar_service(creds: Credentials)` that creates `googleapiclient.discovery.build("calendar", "v3", credentials=creds)`.

   - `def fetch_raw_events(service, target_date: date, calendar_ids: list[str], timezone: str = "America/New_York") -> list[dict]`:
     - For each calendar_id, call `service.events().list()` with:
       - `calendarId=cal_id`
       - `timeMin=start_of_day.isoformat()` (midnight in configured timezone)
       - `timeMax=end_of_day.isoformat()` (next midnight)
       - `singleEvents=True` (expand recurring events)
       - `orderBy="startTime"`
       - `showDeleted=True` (include cancelled events per user decision)
       - Do NOT pass `eventTypes` (omitting returns all types, avoids repeated-parameter bug)
     - Handle pagination: loop on `nextPageToken` until exhausted
     - Tag each event dict with `_calendar_id` = cal_id
     - Return combined list of all raw event dicts

   - `def extract_meeting_link(event: dict) -> str | None`:
     - Check `conferenceData.entryPoints` for type="video" URI first
     - Then check `hangoutLink`
     - Then scan `location` field for URL pattern (http/https)
     - Return first match or None

   - `def normalize_event(event: dict, user_email: str | None = None) -> NormalizedEvent`:
     - Detect all-day vs timed: check `event["start"].get("date")` vs `event["start"].get("dateTime")`
     - Parse times with `dateutil.parser.isoparse()` for timed events
     - Calculate duration_minutes from `(end_dt - start_dt).total_seconds() / 60` for timed events
     - Build Attendee list from `event.get("attendees", [])` with: displayName -> name, email, responseStatus -> response_status (map to ResponseStatus enum), self -> is_self, organizer -> is_organizer
     - Detect recurring: `is_recurring = "recurringEventId" in event`
     - Read event_type from `event.get("eventType", "default")`
     - Read status from `event.get("status", "confirmed")`
     - Extract meeting link via `extract_meeting_link()`
     - Truncate description to first 500 chars if present
     - Store full raw event in `raw_data`
     - Calendar ID from `event.get("_calendar_id", "primary")`

   - `def categorize_events(events: list[NormalizedEvent], user_email: str | None = None) -> dict`:
     Returns dict with keys: `all_day_events`, `timed_events`, `declined_events`, `cancelled_events`
     - Cancelled: status == "cancelled"
     - Declined: not cancelled AND user's attendee entry has response_status == DECLINED (find by is_self=True or matching user_email)
     - All-day: not cancelled, not declined, all_day == True
     - Timed: everything else (confirmed/tentative timed events)
     - Sort timed_events by start_time

   - `def apply_exclusion_patterns(events: list[NormalizedEvent], patterns: list[str]) -> list[NormalizedEvent]`:
     - Filter out events whose title matches any pattern (case-insensitive substring match)
     - Return remaining events

   - `def cache_raw_response(raw_events: list[dict], target_date: date, output_dir: Path) -> Path`:
     - Write JSON to `output_dir/raw/YYYY/MM/DD/calendar.json`
     - Create directories as needed
     - Return path to written file

   - `def fetch_events_for_date(service, target_date: date, config: dict, user_email: str | None = None) -> tuple[dict, list[dict]]`:
     Top-level function that orchestrates: fetch_raw_events -> apply_exclusion_patterns -> normalize_event for each -> categorize_events. Returns (categorized_dict, raw_events_list).
