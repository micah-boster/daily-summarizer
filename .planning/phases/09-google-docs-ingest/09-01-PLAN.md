```python
class SourceType(StrEnum):
    GOOGLE_DOC_EDIT = "google_doc_edit"
    GOOGLE_DOC_COMMENT = "google_doc_comment"

class ContentType(StrEnum):
    EDIT = "edit"
    COMMENT = "comment"

class SourceItem(BaseModel):
    id: str
    source_type: SourceType
    content_type: ContentType
    title: str
    timestamp: datetime
    content: str
    summary: str | None = None
    participants: list[str] = Field(default_factory=list)
    source_url: str
    context: dict = Field(default_factory=dict)
    display_context: str = ""
    raw_data: dict | None = None

    def attribution_text(self) -> str:
        if self.display_context:
            return f"(per {self.display_context})"
        return f"(per {self.source_type.value})"
```

  <name>Task 1: Create Google Docs ingest module with edit detection and content extraction</name>
  <files>src/ingest/google_docs.py, config/config.yaml</files>
  <action>
Create `src/ingest/google_docs.py` with the following functions:

1. **`fetch_google_docs_items(config: dict, creds: Credentials, target_date: date) -> list[SourceItem]`** — Main entry point. Returns empty list if `config["google_docs"]["enabled"]` is False. Calls `_find_edited_docs`, `_build_doc_edit_items`, and `_build_comment_items`. Applies exclusion filtering.

2. **`_get_user_email(drive_service) -> str`** — Call `drive_service.about().get(fields="user(emailAddress)")` to get the authenticated user's email. Cache in module-level variable to avoid repeated API calls.

3. **`_find_edited_docs(drive_service, target_date: date) -> list[dict]`** — Use Drive API `files.list` with query:
   - MIME types: `application/vnd.google-apps.document`, `application/vnd.google-apps.spreadsheet`, `application/vnd.google-apps.presentation`
   - Date filter: `modifiedTime >= '{date}T00:00:00' and modifiedTime < '{next_date}T00:00:00'`
   - `trashed = false`
   - Fields: `id, name, mimeType, modifiedTime, owners, lastModifyingUser`
   - Paginate with `nextPageToken`
   - Filter results to only include docs where `lastModifyingUser.emailAddress` matches authenticated user OR `lastModifyingUser.me` is True (handles both)

4. **`_should_exclude(doc_meta: dict, config: dict) -> bool`** — Check `config["google_docs"]["exclude_ids"]` (list of doc IDs) and `config["google_docs"]["exclude_title_patterns"]` (list of regex patterns). Return True if doc should be skipped.

5. **`_build_doc_edit_items(drive_service, docs_service, edited_docs: list[dict], config: dict) -> list[SourceItem]`** — For each doc:
   - If mimeType is `application/vnd.google-apps.document`: call `_extract_doc_text` from `src/ingest/drive.py` to get content, truncate to `config["google_docs"].get("content_max_chars", 2500)` chars
   - If mimeType is spreadsheet or presentation: content = f"[{file_type} edited on {modifiedTime}]" (metadata only per user decision)
   - Convert to SourceItem with:
     - `id`: doc ID
     - `source_type`: `SourceType.GOOGLE_DOC_EDIT`
     - `content_type`: `ContentType.EDIT`
     - `title`: doc name
     - `timestamp`: parsed `modifiedTime`
     - `content`: extracted/truncated text or metadata string
     - `source_url`: appropriate Google URL based on mimeType (docs.google.com/document/d/{id}, docs.google.com/spreadsheets/d/{id}, docs.google.com/presentation/d/{id})
     - `display_context`: `f"Google Doc {doc_name}"` for Docs, `f"Google Sheet {doc_name}"` for Sheets, `f"Google Slides {doc_name}"` for Slides
   - Wrap each doc in try/except, log warning on failure, continue with remaining docs
   - Respect `config["google_docs"].get("max_docs_per_day", 50)` limit

6. **`_build_comment_items(drive_service, edited_docs: list[dict], target_date: date, config: dict) -> list[SourceItem]`** — For each doc in edited_docs:
   - Call `drive_service.comments().list(fileId=file_id, fields="nextPageToken, comments(id, content, author, createdTime, modifiedTime, resolved, quotedFileContent, replies)", pageSize=100, includeDeleted=False)` with pagination
   - Filter comments where `createdTime` OR `modifiedTime` falls within target_date
   - Include resolved comments (per user decision)
   - For each matching comment, convert to SourceItem:
     - `id`: f"{file_id}_comment_{comment_id}"
     - `source_type`: `SourceType.GOOGLE_DOC_COMMENT`
     - `content_type`: `ContentType.COMMENT`
     - `title`: f"Comment by {author_name} on {doc_name}"
     - `content`: If `quotedFileContent` exists (suggestion): f'Suggested: "{quoted_text}" — {comment_text}'. Otherwise: comment text. Truncate to `config["google_docs"].get("comment_max_chars", 500)` chars.
     - `participants`: [author display name]
     - `source_url`: Google doc URL
     - `display_context`: f"Google Doc {doc_name}"
   - Also include replies that fall within target_date as separate SourceItems
   - Wrap in try/except per doc — if comments API fails (403 on shared docs), log warning and continue

Add `google_docs` section to `config/config.yaml`:
```yaml
google_docs:
  enabled: false
  content_max_chars: 2500
  comment_max_chars: 500
  max_docs_per_day: 50
  exclude_ids: []
  exclude_title_patterns: []
```

Import `logger = logging.getLogger(__name__)` at module level. Use the same logging pattern as `src/ingest/slack.py`.
