1. **src/entity/migrations.py** -- Schema versioning using PRAGMA user_version pattern from research.

- `SCHEMA_VERSION = 1` constant (increment when adding migrations)
- `_get_version(conn: sqlite3.Connection) -> int` -- reads PRAGMA user_version
- `_set_version(conn: sqlite3.Connection, version: int) -> None` -- sets PRAGMA user_version
- `_migrate_v0_to_v1(conn: sqlite3.Connection) -> None` -- creates all 5 tables + indexes using the DDL from RESEARCH.md. Use standard SQL. Keep `strftime()` defaults as safety net but document that Python-generated timestamps are the primary mechanism. Include all indexes from the research DDL.
- `MIGRATIONS: list` -- ordered list: `[_migrate_v0_to_v1]`
- `run_migrations(conn: sqlite3.Connection) -> None` -- checks current version, applies pending migrations in order, sets version after each, commits after each. Assert `len(MIGRATIONS) == SCHEMA_VERSION` at module level to catch ordering bugs.

All sqlite3 imports confined to this module and db.py only (per user decision about repository module isolation -- migrations.py is part of the entity package's DB layer).

2. **src/entity/db.py** -- Database connection factory.

- `get_connection(db_path: str, auto_create: bool = True) -> sqlite3.Connection` -- Opens a connection to the given path. If auto_create is True and the parent directory does not exist, creates it. Sets `PRAGMA journal_mode=WAL` (verify return value is "wal"). Sets `PRAGMA foreign_keys=ON`. Sets `conn.row_factory = sqlite3.Row`. Calls `run_migrations(conn)`. Returns the connection. If the file does not exist and auto_create is False, raises FileNotFoundError.

- `get_connection_from_config(config: EntityConfig) -> sqlite3.Connection | None` -- Wraps get_connection with graceful degradation. If config.enabled is False, returns None. If any exception occurs (file permissions, corrupt DB, etc.), logs a warning and returns None. This is the entry point for pipeline integration.

Import only sqlite3 and pathlib in this module. Import run_migrations from src.entity.migrations.
