**Step 3: Backfill orchestrator** (src/entity/backfill.py)

Create `run_backfill(from_date, to_date, config, force=False, client=None)`:

1. Open EntityRepository from config.entity.db_path
2. Generate list of dates in range
3. For each date, check if already processed in backfill_progress table (skip unless force=True)
4. Group unprocessed dates into weekly batches (7 days per batch)
5. For each batch:
   a. For each day in batch:
      - Try to read the daily sidecar JSON from `output/daily/YYYY/MM/YYYY-MM-DD.json`
      - If sidecar exists and has content, extract synthesis text from it (substance, decisions, commitments sections)
      - If sidecar is empty or missing, try to read the daily markdown from `output/daily/YYYY/MM/YYYY-MM-DD.md`
      - If markdown exists, use the raw text as input (per CONTEXT.md: "send synthesis text through entity extraction prompt")
      - If neither exists, mark as 'skipped' in backfill_progress and continue
      - Call `extract_entities(text, config, client)` on the content
      - For each extracted entity with confidence >= config.entity.auto_register_threshold:
        - Normalize the name using `normalize_for_matching`
        - Check if entity exists in registry (resolve_name or get_by_name)
        - If not found, register new entity
        - If raw name differs from normalized canonical, add raw name as alias
      - Record day as 'completed' in backfill_progress with entity count
   b. After all days in batch complete, commit the checkpoint (all or nothing per batch)
   c. Print progress: "Batch N/M complete: processed X days, found Y entities"
6. Print final summary: "Backfill complete: processed X/Y days, Z entities registered"

Progress display format (per CONTEXT.md): "Processing 2025-10-01 to 2026-04-05... [###---] 120/180 days, 47 entities found"

Use simple f-string with `\r` carriage return for progress (per RESEARCH.md: no need for tqdm/rich).

Helper functions:
- `_read_day_content(date, output_dir) -> str | None` — reads sidecar JSON or markdown for a date
- `_extract_text_from_sidecar(sidecar_data: dict) -> str` — concatenates substance/decisions/commitments text from sidecar JSON structure
- `_is_day_processed(conn, date_str) -> bool` — checks backfill_progress table
- `_record_day_processed(conn, date_str, status, entities_found)` — inserts into backfill_progress

```python
def _discover_and_register_entities(
    synthesis_result: dict,
    target_date: date,
    config: PipelineConfig,
) -> None:
    """Post-synthesis entity discovery. Fire-and-forget with graceful degradation."""
    if not config.entity.enabled:
        return

    try:
        from src.entity.db import get_connection_from_config
        from src.entity.normalizer import normalize_for_matching
        from src.entity.repository import EntityRepository

        conn = get_connection_from_config(config)
        if conn is None:
            return

        # Collect entity names from synthesis items
        all_entity_names: set[str] = set()
        for section_key in ("substance", "decisions"):
            for item in synthesis_result.get(section_key, []):
                if hasattr(item, "entity_names"):
                    all_entity_names.update(item.entity_names)
                elif isinstance(item, dict) and "entity_names" in item:
                    all_entity_names.update(item["entity_names"])
        for commitment in synthesis_result.get("commitments", []):
            if hasattr(commitment, "entity_names"):
                all_entity_names.update(commitment.entity_names)
            elif isinstance(commitment, dict) and "entity_names" in commitment:
                all_entity_names.update(commitment["entity_names"])

        if not all_entity_names:
            logger.debug("No entity names in synthesis output")
            conn.close()
            return

        repo = EntityRepository(config.entity.db_path)
        with repo:
            registered = 0
            for raw_name in all_entity_names:
                if not raw_name or not raw_name.strip():
                    continue
                normalized = normalize_for_matching(raw_name)
                existing = repo.resolve_name(normalized)
                if existing is None:
                    # Also try the raw name
                    existing = repo.resolve_name(raw_name)
                if existing is None:
                    # Determine type heuristic: names with 2+ words where each word
                    # is capitalized and short are likely people
                    parts = raw_name.strip().split()
                    is_person = (
                        len(parts) >= 2
                        and all(p[0].isupper() for p in parts if p)
                        and all(len(p) <= 20 for p in parts)
                        and len(parts) <= 4
                    )
                    entity_type = "person" if is_person else "partner"
                    repo.add_entity(raw_name.strip(), entity_type)
                    registered += 1
                    # Add normalized form as alias if different
                    if normalized != raw_name.strip().lower():
                        try:
                            from src.entity.normalizer import normalize_company_name
                            canonical = normalize_company_name(raw_name.strip())
                            if canonical != raw_name.strip():
                                entity = repo.get_by_name(raw_name.strip())
                                if entity:
                                    repo.add_alias(entity.id, canonical)
                        except Exception:
                            pass  # Alias is best-effort

            if registered:
                logger.info("Entity discovery: registered %d new entities", registered)
        conn.close()

    except Exception as e:
        logger.warning("Entity discovery failed: %s. Daily summary unaffected.", e)
```
