  <files>src/config.py, src/synthesis/extractor.py</files>
  <action>
1. In `src/config.py`, add `max_concurrent_extractions: int = Field(default=3, ge=1, le=10)` to the `SynthesisConfig` class, after the existing `monthly_max_output_tokens` field. This controls the asyncio.Semaphore value for parallel Claude calls.

2. In `src/synthesis/extractor.py`, add two new async functions BELOW the existing sync functions (do not modify existing sync functions):

**`extract_meeting_async()`**: Async version of `extract_meeting`. Accepts `anthropic.AsyncAnthropic` client instead of `anthropic.Anthropic`. Uses `await client.messages.create(...)` with the same structured output parameters (output_config with json_schema). Include the same fallback pattern for older SDK (try output_config, catch TypeError/BadRequestError, fall back to extra_headers/extra_body beta format). Apply the existing `retry_api_call` decorator to the inner async helper functions (tenacity auto-detects async coroutines and awaits them). Reuse `_convert_output_to_extraction()` for output conversion (it's sync and doesn't need changing). Signature: `async def extract_meeting_async(event: NormalizedEvent, config: PipelineConfig, client: anthropic.AsyncAnthropic) -> MeetingExtraction | None`.

**`extract_all_meetings_async()`**: Async version of `extract_all_meetings`. Creates an `asyncio.Semaphore(config.synthesis.max_concurrent_extractions)`. For each event with a transcript, creates a semaphore-guarded task. Uses `asyncio.gather(*tasks, return_exceptions=True)` to run all concurrently. Iterates results: if `isinstance(r, BaseException)`, logs warning and increments skipped count; if `r is not None`, appends to extractions list. Logs same stats as sync version (extracted count, low-signal count, failed count). Signature: `async def extract_all_meetings_async(events: list[NormalizedEvent], config: PipelineConfig, client: anthropic.AsyncAnthropic) -> list[MeetingExtraction]`.

Add `import asyncio` at the top of extractor.py.
