<task type="auto">
  <name>Task 2: Build Notion ingest module with API client, page/DB fetching, and SourceItem conversion</name>
  <files>src/ingest/notion.py</files>
  <action>
Create src/ingest/notion.py following the google_docs.py pattern. The module needs:

**NotionClient class** (thin httpx wrapper with rate limiting):
- __init__(token, notion_version="2022-06-28"): Set up headers (Authorization: Bearer {token}, Notion-Version, Content-Type: application/json), base URL "https://api.notion.com/v1", last_request timestamp, min_interval=0.35
- _throttle(): Sleep if less than min_interval since last request (use time.monotonic)
- _request(method, path, **kwargs): Throttle, make httpx request, raise_for_status, return .json()
- _paginate(method, path, **kwargs): Iterate with has_more/start_cursor pagination, yield all results
- Wrap _request calls with the existing @retry_api_call decorator for 429/5xx retries

**_get_bot_user_id(client) -> str**: Call GET /users/me, return bot user id. Cache at module level (same pattern as google_docs._user_email_cache).

**_fetch_edited_pages(client, target_date, config) -> list[SourceItem]**:
- POST /search with filter {"property": "object", "value": "page"}, sort by last_edited_time descending
- Paginate through all results
- Filter client-side: last_edited_time falls within target_date (>= start of day, < start of next day, ISO format comparison)
- For each matching page: fetch block children (GET /blocks/{page_id}/children?page_size=50) to extract text content
- Extract text using _extract_text_from_blocks() -- supported block types: paragraph, heading_1, heading_2, heading_3, bulleted_list_item, numbered_list_item, toggle, callout, quote. For each, extract rich_text[].plain_text joined.
- Truncate content to config.notion.content_max_chars
- Convert to SourceItem with:
  - id=page_id
  - source_type=SourceType.NOTION_PAGE
  - content_type=ContentType.EDIT
  - title=page title (from properties.title or properties.Name, handling rich_text array)
  - timestamp=dateutil_parse(last_edited_time)
  - content=extracted text (truncated)
  - source_url=page url
  - display_context=f"Notion {page_title}" (for "(per Notion Page Title)" attribution)
- Respect max_pages_per_day limit, log warning if exceeded

**_extract_page_title(page: dict) -> str**: Extract title from page properties. Notion pages have a "title" type property (the name varies). Iterate page["properties"], find the one with type=="title", extract rich_text[].plain_text. Fallback to "Untitled".

**_extract_text_from_blocks(blocks: list[dict]) -> str**:
- SUPPORTED_BLOCKS = {"paragraph", "heading_1", "heading_2", "heading_3", "bulleted_list_item", "numbered_list_item", "toggle", "callout", "quote"}
- For each block, if type in SUPPORTED_BLOCKS, extract block[type]["rich_text"] -> join plain_text values
- Join all parts with newline
- Return joined text

**_fetch_database_changes(client, target_date, config) -> list[SourceItem]**:
- For each database_id in config.notion.watched_databases:
  - POST /databases/{database_id}/query with filter: {"timestamp": "last_edited_time", "last_edited_time": {"on_or_after": "{date}T00:00:00", "before": "{next_date}T00:00:00"}}
  - Paginate through results
  - For each DB item: extract property values using _extract_db_properties()
  - Also fetch database metadata (GET /databases/{database_id}) for the database title
  - Convert to SourceItem with:
    - id=item_id
    - source_type=SourceType.NOTION_DB
    - content_type=ContentType.STAGE_CHANGE
    - title=item title (from title property)
    - timestamp=dateutil_parse(last_edited_time)
    - content=formatted property summary (e.g., "Status: Done, Priority: High, Assignee: Micah")
    - source_url=item url
    - display_context=f"Notion {database_title}" (for "(per Notion DB Name)" attribution)
- Respect max_db_items_per_day limit per database

**_extract_db_properties(properties: dict) -> str**:
- Iterate properties, extract human-readable values for common types:
  - "title": rich_text[].plain_text
  - "rich_text": rich_text[].plain_text
  - "select": select.name
  - "multi_select": join names with ", "
  - "status": status.name
  - "number": str(number)
  - "date": date.start
  - "checkbox": "Yes"/"No"
  - "people": join people[].name with ", "
  - "url": url value
  - Skip: relation, rollup, formula, files, created_by, last_edited_by, created_time, last_edited_time (too noisy)
- Format as "Key: Value" pairs joined with ", "
- Truncate to config.notion.content_max_chars

**fetch_notion_items(config: PipelineConfig, target_date: date) -> list[SourceItem]** (public entry point):
- If not config.notion.enabled: return []
- If not config.notion.token: log warning, return []
- Build NotionClient with config.notion.token and config.notion.notion_version
- Call _fetch_edited_pages + _fetch_database_changes
- Log totals
- Return combined list
